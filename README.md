### 安装方法

解压 zip，得到 network-sniffer/ 文件夹
打开 Chrome → 地址栏输入 chrome://extensions/
右上角开启 开发者模式
点击 加载已解压的扩展程序 → 选择 network-sniffer/ 文件夹
点击浏览器右上角的插件图标，侧边栏会打开


### 使用方式
操作效果在页面选中文字自动在侧边栏高亮匹配的请求点击请求查看完整请求/响应详情，匹配文字会标黄切换 Tab过滤全部 / 仅匹配 / Fetch / XHR输入 URL 过滤按路径快速定位点击「匹配」手动刷新选中文本点击「清空」清除已记录的请求
匹配逻辑：

🔵 内容匹配 — 响应体或请求体包含选中文字
🟡 时间匹配 — 选中后 3 秒内触发的请求
🟣 双重匹配 — 两者都满足，最可能是目标接口
